# NB-keil

### 简介

用于编译合适的NB驱动代码，基于STM32F103平台开发


---

## 组成

- keil 中为keil v5的工程文件
- Net 中是NB的相关驱动文件
- Protocol 中是上传的协议文件
- Hal中是外设传感器的驱动文件
