# BC95-keil

### 项目描述

移远BC95，作为最早上市的NB模组，采用海思芯片，被广泛研究和应用。


---

## 组成

- keil 中为keil v5的工程文件
- Net 中是NB的相关驱动文件
- Protocol 中是上传的协议文件
- Hal中是外设传感器的驱动文件
