# freeRTOS(NB-IoT)

## 简介

使用NB进行通信的freeRTOS工程模板，基于STM32F103平台开发


---

## 组成

#### BC26

基于联发科全网通NB-IoT

#### BC28

华为海思全网通NB-IoT

#### BC95

最早上市的NB-IoT

---


- keil 中为keil v5的工程文件
- Net 中是NB的相关驱动文件
- Protocol 中是上传的协议文件
- Hal中是外设传感器的驱动文件
